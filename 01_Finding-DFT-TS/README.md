Lesson 01: Finding transition states using DFT
=======================

The purpose of this lesson is to walk students through calculating reaction and product energies and finding transition states and energies for S<sub>N</sub>1, S<sub>N</sub>2, E1, and E2 reaction pathways.

## Lesson Information
### Lesson Learning Outcomes
At the end of this lesson, students will be able to...
1. Draw moleculear geometries in ChimeraX and save them as *.xyz files.
2. Generate and save an ORCA input file using ChimeraX.
3. Run an ORCA input file to generate an output file.
4. Do basic analysis of an ORCA output file using either ChimeraX or a text editor.
5. Determine reaction energies in Hartree kcal/mol, and kJ/mol.

### Cyberinfrastructure Prerequisites

No cyberinfrastructure skills are required for this lesson.

### Content Prerequisites

Before beginning this lesson, students are expected to be familiar with the
following content topics:

* Thermochemistry (first-year general chemistry)
* Chemical kinetics (first-year general chemistry)
* S<sub>N</sub>1, S<sub>N</sub>2, E1, E2 reactions (second-year organic chemistry - optional) 

### Resources

* ChimeraX (https://www.cgl.ucsf.edu/chimerax/)
* ORCA (https://www.faccts.de/orca/)
